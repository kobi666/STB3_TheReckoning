Manual:
-----------------------------------------------------
1) To create a spline chose GameObjecty->Create Other->BG Curve or AddComponent and search for "BGCurve".
2) Select Components tab, hit plus icon and add Collider3DBox (for 3D). Collider2DBox or Collider2DEdge (for 2D).
3) Set up snapping if you need it (under Points tab) 
4) Set 2D Mode to XY if you use 2D components
5) Add points
6) Adjust component parameters
